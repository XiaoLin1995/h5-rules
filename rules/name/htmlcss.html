# HTML/CSS文件命名

确保文件命名总是以字母开头而不是数字，且字母一律小写，以下划线连接且不带其他标点符号，如：

	<!-- HTML -->
	jdc.html
	jdc_list.html
	jdc_detail.html

	<!-- SASS -->
	jdc.scss
	jdc_list.scss
	jdc_detail.scss
